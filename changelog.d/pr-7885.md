### 🏠 Internal

- Pin `asv-runner<0.2.2` in `.github/workflows/benchmarks.yml`.  Starting with asv-runner 0.2.2 (released 2026-06-27), `do_setup_cache()` invokes any class-level `setup(self)` method it can bind with no positional args -- for the `save_run` benchmarks that means `setup()` runs before `setup_cache()` has written `bm_heavy.tar`, breaking every Benchmarks CI run since #7402.  Reported upstream at [airspeed-velocity/asv_runner#52](https://github.com/airspeed-velocity/asv_runner/issues/52).  [PR #7885](https://github.com/datalad/datalad/pull/7885) (by [@yarikoptic](https://github.com/yarikoptic))
